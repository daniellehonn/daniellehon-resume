PDF to MD Converter

Open Source

A powerful PDF to Markdown Converter with advanced LLM OCR capabilities, specifically designed to handle complex documents with precision and accuracy. This tool goes beyond traditional parsing by incorporating AI-driven OCR, enabling the extraction of text from scanned PDFs and images with exceptional clarity.

Key features include:

  • Advanced LLM OCR for accurate text extraction from scanned PDFs and images
  • Support for complex document structures, including tables, formulas, filled forms, and multi-column layouts
  • Customizable output formats, including JSON and plain Markdown text

Created an SDK package for this project to make it more easily accessible to the public

PDF Parser Test Suite

Open Source

An open-source testing suite designed to rigorously evaluate and compare the performance of leading PDF parsers:

  • LlamaParse, ContextForce, and Jina.ai.

This comprehensive suite benchmarks each parser's capabilities across key areas:

  • Basic text extraction
  • Markdown header extraction
  • OCR extraction
  • Table extraction
  • Formula detection

By utilizing a diverse set of test PDFs, the suite provides in-depth insights into each parser's strengths and weaknesses, offering developers and researchers a valuable tool for optimizing and selecting the best parser for their specific needs.
